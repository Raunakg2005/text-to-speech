## ğŸ¯ Usage

1. **Enter Your Script** - Type or paste your narrative text into the text area
2. **Select a Voice** - Choose from available system voices in the dropdown
3. **Adjust Settings** - Fine-tune speed (0.5x - 2x), pitch (0.5 - 2), and volume (0% - 100%)
4. **Click Speak** - Hit the speak button to start narration
5. **Control Playback** - Use pause, resume, or stop buttons as needed
